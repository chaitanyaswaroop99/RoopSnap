# How to Change Website Name

## Option 1: Change Site Title/Branding (In Code)

To change "RoopSnap" to a different name throughout the website:

### Files to Update:
1. **index.html** - Line 6 (title), Line 16 (logo), Line 35 (hero title)
2. **admin.html** - Title and headers
3. **README.md** - Documentation
4. **package.json** - Project name

### Quick Find & Replace:
Search for: `RoopSnap`
Replace with: `Your New Name`

---

## Option 2: Change Domain/URL (On Platform)

### Railway:
1. Go to your project
2. Click **Settings** → **Domains**
3. Click **Generate Domain** (for new random name)
4. Or add **Custom Domain**

### Render:
1. Go to your service
2. Click **Settings**
3. Change **Service Name** (changes URL)
4. Or add **Custom Domain**

### Netlify:
1. Go to **Site settings**
2. Click **Change site name**
3. Or add **Custom domain**

---

## When to Change:

✅ **Before Deployment**: Edit code, then deploy  
✅ **During Deployment**: Change domain in platform settings  
✅ **After Deployment**: Update code and redeploy, or change domain anytime
